What is the Enye Ñ and ñ Character?

The enye character (Ñ/ñ) is the 15th letter of the Spanish alphabet, created by placing a tilde (~) over the letter "n." This diacritical mark fundamentally changes the sound from a regular "n" to a palatalized "ny" sound, similar to the "ni" in "onion."

Historically, the enye evolved from medieval scribes who wrote "nn" as a shorthand. The character is crucial for proper spelling in languages like Spanish, Filipino/Tagalog, Galician, and Asturian.

How to Type Enye on Windows Keyboard (Alt Codes + Shortcuts)

For Windows users, here are the easiest Enye shortcut key combinations:

Method 1: Alt Codes for Enye

The Alt code for enye is the most widely used method on Windows computers with a dedicated keypad. Make sure your Num Lock is active first:

  • For uppercase Ñ: Hold the Alt key, type 165 or 0209 on the numeric keypad, and release Alt.
  • For lowercase ñ: Hold the Alt key, type 164 or 0241 on the numeric keypad, and release Alt.

Method 2: US-International Keyboard Layout

Go to Settings > Time & Language > Language. Add the "United States-International" keyboard layout. Switch layouts using Win + Spacebar. Once enabled:

  • Type right Alt + n for ñ or right Alt + Shift + n for Ñ.

Method 3: Character Map Tool

Press Win + R, type charmap, and hit Enter. Locate the Enye character, click "Select" and then "Copy" to paste it anywhere.
