# Enye Copy Paste (Ñ/ñ) — Special Character Tool

A fast, lightweight, and mobile-friendly online utility tool to copy and paste the Spanish **Enye (Ñ or ñ)** character with a single click.

## 🚀 Live Tool
You can access the live web utility here:
👉 **[https://enyecopypaste.com](https://enyecopypaste.com/)**

---

## 🛠️ Features
- **One-Click Copy:** Click on Ñ or ñ to instantly copy it to your clipboard.
- **Keyboard Guides:** Easy-to-follow tutorials on how to type Enye on Windows, Mac, and mobile devices.
- **100% Free & Fast:** No installation or registration required. Fully optimized for speed.
